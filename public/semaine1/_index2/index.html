Présentation du cours et l'environnement de travail

Objectifs pédagogiques

À la fin de la séance, vous devriez être capables de :

  • Comprendre les objectifs du cours et les règlements du collège à respecter (Quiz WooClap)
  • Vous connecter à votre compte OneDrive scolaire.
  • Naviguer dans l’explorateur de fichiers Windows 11 (créer, nommer, organiser).
  • Ouvrir un dossier de projet dans VS Code.
  • Créer un fichier .ipynb avec VS Code (Jupyter Notebook).
  • Écrire et exécuter du code Python simple dans un notebook.
  • Sauvegarder leur travail et comprendre l’organisation de leurs fichiers.

1. Présentation du cours (~ 40 minutes)

  • Pondération et place du cours dans le programme
  • Méthode pédagogique: déroulement des séances de cours et présentation des calendriers et des plateformes utilisées

Pause 5 minutes

Prise en main de l’environnement de travail - Activité guidée (~ 120 minutes)

Étape 1 – Organisation des fichiers (15 min)

But : créer un espace de travail simple et structuré

Sachez que..

Vous pouvez changer les noms des dossiers suggérés, en autant que vous utilisiez des noms représentatifs.

  1. Ouvrir l’explorateur de fichiers Windows.

La combinaison de touches Windows + E permet d’ouvrir l’Explorateur de fichiers dans Windows. C’est un raccourci clavier très pratique pour accéder rapidement à vos fichiers et dossiers.

  1. Créer un dossier dans Documents appelé Python-Sciences.
  2. Dans ce dossier, créer un sous-dossier Semaine01-Environnement.
  3. Ouvrir VS Code.
  4. Cliquer sur Fichier > Ouvrir le dossier… et choisir Semaine01-Environnement.

💡 Explication: Pourquoi on travaille dans un dossier bien identifié ? Pour organiser ses projets et éviter de perdre ses fichiers.

🧠 Étape 2 – Découverte de VS Code (15 min)

But : repérer les éléments essentiels de l’interface

  1. Identifier dans VS Code :

    • L’explorateur de fichiers (icône dossier 📁).
    • La barre latérale (explorateur, recherche, extensions…).
    • Le bouton de terminal (Ctrl + `).
    • Le panneau de fichiers ouverts (onglets).
    • Le bouton Nouveau fichier → créer un fichier test.txt et l’ouvrir.

  2. Installer les extensions nécessaires (si non déjà faites) :

    • Python
    • Jupyter

💬 Pause discussion rapide :

  • À quoi sert chaque panneau ?
  • Qu’est-ce qu’un terminal ? Pourquoi c’est utile en programmation ?

📓 Étape 3 – Créer et utiliser un Jupyter Notebook (30 min)

But : créer et exécuter un notebook Python

  1. Cliquer sur Nouveau fichier → nommer premier_notebook.ipynb.

  2. Dans la première cellule :

    # Mon premier programme Python
print("Bonjour tout le monde !")

    Puis cliquer sur Exécuter (bouton ▶️).

  3. Ajouter une cellule Markdown :

    # Semaine 1 – Environnement de développement
Voici mon premier notebook avec du code et des commentaires.

  4. Créer une cellule de calcul :

    3 * (4 + 2)

  5. Sauvegarder avec Ctrl + S. Vérifier que le fichier .ipynb apparaît bien dans le dossier.

🧠 Mini-défis (50 min)

🎯 Objectif : Consolider en autonomie les manipulations de base

Distribuer une fiche de défis à faire dans leur notebook :

  1. 🗂️ Créer un nouveau notebook exercice_fichiers.ipynb dans un sous-dossier exercices.

  2. Dans une cellule Markdown, expliquer la différence entre Markdown et Python dans un notebook.

  3. Écrire un programme qui affiche :

    nom = "Camille"
print("Bonjour", nom)

  4. Faire un calcul scientifique :

    masse = 2.5  # kg
acceleration = 9.8  # m/s²
force = masse * acceleration
print("Force =", force, "N")

  5. Sauvegarder, fermer, puis réouvrir le fichier .ipynb pour vérifier que tout fonctionne.

🔁 Retour réflexif (20 min)

Discussion ouverte :

  • Avez-vous rencontré des problèmes ?
  • Qu’est-ce qui vous semble encore flou ?
  • Que fait exactement un notebook quand on l’exécute ?

Bonus : montrer où se trouve le fichier .ipynb sur le disque, l’ouvrir avec Notepad pour montrer que c’est du JSON (mais illisible tel quel) → insister sur l’intérêt de garder ses fichiers bien rangés.
