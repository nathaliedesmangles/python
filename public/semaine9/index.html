Option 1 : Utiliser np.genfromtxt() (plus robuste)

Si ton fichier contient :

  • une premi√®re ligne d‚Äôen-t√™tes,
  • des valeurs manquantes,
  • ou des colonnes non num√©riques (comme des dates),

Lecture uniquement de la colonne num√©rique :

Explications :

  • skip_header=1 : ignore la premi√®re ligne contenant les titres.
  • usecols=1 : lit seulement la colonne d‚Äôindice 1 (ici, les temp√©ratures).
  • np.genfromtxt() : accepte les fichiers mixtes (texte + nombres) et les valeurs manquantes.

Exemple avanc√© avec plusieurs types de donn√©es
