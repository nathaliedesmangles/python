Installation & premier lancement

  1. Télécharge VS Code depuis code.visualstudio.com (bouton Download for Windows) (code.visualstudio.com).
  2. Exécute l’installateur, coche Add to PATH, puis installe.
  3. Lance VS Code via le menu Démarrer — tu devrais voir l’écran d’accueil (“Welcome”) .

Interface principale

  • Barre d’activité (gauche) : accès à l’explorateur, Git, extensions…
  • Volet Explorateur : tes fichiers/sous-dossiers.
  • Éditeur : là où tu écris ton code.
  • Barre d’onglets : navigation entre fichiers ouverts.
  • Terminal intégré (en bas).
  • Barre d’état : info sur la branche Git, format du fichier, erreurs, etc.

Les captures d’écran en début de guide montrent ces zones dans l’interface (code.visualstudio.com).

Ouvrir ou créer un projet

  1. Fichier → Ouvrir un dossier… (Ctrl+K Ctrl+O) (code.visualstudio.com).
  2. Crée ou sélectionne un dossier (ex. mon-projet), puis clique Sélectionner un dossier.
  3. Le dossier s’affiche dans l’Explorateur, prêt à être utilisé .

Créer & éditer des fichiers

  • Clique droit sur le volet Explorateur → Nouveau fichier → nomme-le (index.html, script.py, etc.).
  • Le voici ouvert dans l’éditeur.
  • Les captures montrent l’éditeur avec du code actif (reddit.com).

Fonctionnalités utiles

  • Multi-volets : divise l’éditeur pour deux fichiers côte à côte (Ctrl+\) .
  • Minimap : aperçu rapide du fichier en marge droite (code.visualstudio.com).
  • Sticky Scroll : montre la section de code active en haut de l’éditeur (code.visualstudio.com).
  • Recherche dans le projet (Ctrl+Shift+F), terminal (Ctrl+\``), **Git** (Ctrl+Shift+G), **extensions** (Ctrl+Shift+X`) .

Astuces clavier & productivité

  • Glisser-déposer fichiers pour les ouvrir.
  • Commenter : Ctrl+/.
  • Aller à une ligne : Ctrl+G.
  • Formatage : Ctrl+K Ctrl+F.
  • Renommer symboles : F2.
  • IntelliSense : suggestions automatiques avec Ctrl+Space (code.visualstudio.com, code.visualstudio.com).

Extensions utiles

  • Python (Microsoft) : exécution et débogage.
  • Live Server : hébergement local HTML.
  • Prettier : formatage automatique.
  • GitLens : suivi avancé Git.
  • CodeSnap : capture de code (pratique pour documenter ou partager) (dev.to, code.visualstudio.com).
