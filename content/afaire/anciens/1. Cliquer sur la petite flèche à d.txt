1. Cliquer sur la petite flèche à droite du menu des applications dans la barre des tâches de Windows afin de faire apparaître les applications masquées.

2. Effectuer un clic droit sur le logo de OneDrive (s'il n'est pas connecté, il apparaît grisé et barré).

![Trouver l'icone](onedrive/1_trouver_onedrive.png?width=30vw)

3. Cliquer sur **« Se connecter »**.

![Se connecter](onedrive/2_se_connecter.png?width=25vw)

4. Entrer votre nom d'utilisateur ***7 derniers chiffres de votre numéro DA*** suivi de `@cmontmorency.qc.ca` et cliquer sur **« Se connecter »**.

![Configurer OneDrive](onedrive/3_configurer_onedrive.png?width=30vw)

5. Si vous avez préalablement créé un compte personnel Microsoft en utilisant votre adresse courriel montmorency, la fenêtre suivante va s'afficher. Il faudra alors choisir l'option **« Compte professionnel ou scolaire »**.

![Compte scolaire](onedrive/5_onedrive_scolaire.png?width=30vw)

6. Entrer votre mot de passe (la fenêtre de connexion doit afficher le logo ou la mention **montmorency** — si ce n’est pas le cas, reprendre la procédure depuis le début).

![Entrer mot de passe](onedrive/6_mot2passe.png?width=30vw)

7. Vous avez la possibilité de choisir l’emplacement de stockage des fichiers synchronisés localement par OneDrive.
   Si l’emplacement proposé par défaut ne vous convient pas, cliquer sur **« Modifier l’emplacement »** pour en choisir un autre.
   Pour poursuivre, cliquer sur **« Suivant »**.

![Choisir dossier de sauvegarde](onedrive/7_choisir_dossier.png?width=30vw)


8. Vous avez la possibilité de synchroniser les fichiers et dossiers présents sur votre **bureau**, dans vos dossiers **Documents** et **Images**.
   Cette configuration est définie par défaut. Pour **ne pas** synchroniser ces dossiers dans OneDrive :

   * Cliquer sur les icônes représentant ces dossiers afin de les désélectionner.
   * Ces paramètres pourront être modifiés plus tard.
     Une fois vos choix effectués, cliquer sur **« Continuer »** ou sur **« Ignorer »** si vous avez désélectionné les trois dossiers.

![Sauvegarde](onedrive/8_sauvegarde_dossier.png?width=30vw)

9. Lire les informations, puis cliquer sur **« Suivant »** dans les **trois fenêtres suivantes**.

![Découvrir OneDrive](onedrive/9_1_decouvrir.png?width=30vw)

![Partager des fichiers](onedrive/9_2_partager.png?width=30vw)

10. Vous avez la possibilité d’installer OneDrive sur votre téléphone mobile :

    * Si vous souhaitez le faire, cliquer sur **« Télécharger l'application mobile »**.
      Une page web va s’ouvrir dans votre navigateur par défaut. Il vous suffira de suivre les instructions pour procéder à l’installation.
    * Si vous ne souhaitez pas utiliser l’application mobile ou préférez la configurer plus tard, cliquer sur **« Plus tard »**.

![Télécharger l'appli](onedrive/10_telecharger_appli.png?width=30vw)

11. Cliquer sur **« Ouvrir mon dossier OneDrive »**.

![Ouvrir dossier OneDrive](onedrive/11_ouvrir_dossier.png?width=30vw)

12. L’explorateur de fichiers va s’ouvrir dans le dossier contenant les fichiers synchronisés, et la synchronisation va commencer.

> **OneDrive est maintenant prêt à être utilisé !**
